Uni-Run Introduction

Uni-Run is a web-based Marathon and Virtual Run Coordinator system developed for students and staff of Universiti Teknologi MARA Shah Alam. The system is designed to simplify the management of campus fun runs and virtual marathon events by providing an online platform for event registration, participant management, and performance tracking. The website allows users to register accounts, join running events, and view event information easily through a user-friendly interface. Administrators can manage participant records, assign bib numbers, update finisher times, and monitor event activities efficiently. In addition, the system includes a dashboard feature that presents important statistics such as total participants, total distance covered, and participant demographics. This project aims to promote a healthy lifestyle among the UiTM Shah Alam community while improving the efficiency of event management through digital technology.

Project Objectives

01

Efficiency

To develop a web-based system for managing marathon and virtual run events at Universiti Teknologi MARA Shah Alam efficiently and systematically.

02

Information Management

To enable administrators to perform Create, Read, Update, and Delete (CRUD) operations for event and participant information.

03

Security

To provide an online registration platform with proper validation and secure login functionality for participants.

04

Data Visualization

To provide a dashboard that displays event statistics and participant data using graphical representations.

05

Healthy Lifestyle Awareness

To encourage student participation in healthy and active lifestyle activities through organized running events.

Project Scope

User Scope

  • • User Registration
    Users can create a new account by filling in their personal information with proper validation.
  • • Login and Logout
    Users can securely login and logout from the system using session management.
  • • Event Participation
    Users can view available marathon or virtual run events and register for their preferred events.
  • • Profile Management
    Users can view and update their personal information and registration details.

Admin Scope

  • • User Management
    Administrators can manage user information stored in the database.
  • • Event Management
    Administrators can create, read, update, and delete marathon or virtual run events.
  • • Participant Management
    Administrators can manage participant registrations, assign bib numbers, and update finisher times.
  • • Data Management
    Administrators can manage and monitor all system data including events, users, and running records.
  • • Dashboard and Reporting
    Administrators can access dashboard analytics and graphical reports related to participant statistics, gender distribution, age categories, and total distance covered.
