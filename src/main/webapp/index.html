01. Introduction

Redefining Campus Athletics.

Uni-Run is a cutting-edge Marathon & Virtual Run Coordinator. We bridge the gap between physical track events and remote fitness tracking, providing a unified platform for students to compete, track, and celebrate fitness milestones in a session-secured environment built on Java MVC.

02. Core Objectives

/01

Seamless event lifecycle management from sign-up to bib assignment.

/02

Real-time community telemetry via high-impact data visualization.

/03

Robust security using server-side session management.

/04

Centralized result logging for administrative transparency.

Project Scope

Runners

Profile management, event discovery, virtual time submission.

Administrators

Full CRUD on events, bib batching, and verified result approval.
