UNIRUN

Phase 1: Concept

Modernizing Campus Athletics Through Digital Innovation.

Uni-Run merupakan platform pengurusan marathon bersepadu yang direka khas untuk komuniti universiti. Menggunakan seni bina Java MVC, sistem ini memastikan integriti data dan prestasi yang stabil bagi menguruskan ribuan peserta dalam satu masa.

Automated BIB Assignment

Penjanaan nombor BIB secara automatik sebaik sahaja pendaftaran disahkan melalui sistem.

Real-time Analytics

Papan pemuka pentadbir yang memaparkan statistik penyertaan dan pencapaian secara langsung.

Project Scope

Module Runners

  • • Event Registration

  • • Result Verification Audit
  • • Participant Analytics
